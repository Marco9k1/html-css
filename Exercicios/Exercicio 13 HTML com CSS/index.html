Usando HTML junto com as CSS

Embora seja errado usar as CSS direto no HTML, ainda da pra fazer algumas coisas por aqui para poupar tempo.

Para mudar a aparencia inicial do site, pode ser acrescentado a tag STYLE com a propiedade BACKGROUND COLOR para mudar cor de fundo.

Também da pra mudar a cor somente das letras acrescenta-se antes da palavra selecionada a tag STYLE com a propriedade COLOR e escolhendo a cor

Mudar a fonte da pagina é com a tag FONTFAMILY
