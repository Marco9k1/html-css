Personalizando links

Pra começar a estilizar os links, posso usar as tags a:visited para mudar a aparencia do link quando ja tiver sido visitado, hover com a configuração text-decoration: underline tira o sublinhado do link, active é pra mudar a cor do link quando for clicado, a::after é uma opção que adiciona algo logo após o link. E cirando uma classe pro link, da pra adicionar um background-color nele ou caracteres e quaiquer configurações desejadas
