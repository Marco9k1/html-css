Para criar essa quadrado de exemplo. Primeiro abri uma tag Style dentro do cabeçalho, depois criei a div. Então pra fazer ficar dentro de um "quadrado" usei as propriedades background-color para escolher a cor dos quadrados, display: inline-block para criar divizões nos quadrados, coloquei uma borda com border: 1px solid e black, alinhei no centro com text-aling, e coloquei a largura e o comprimento com as tags height (largura) e width (comprimento)
