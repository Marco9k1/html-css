Tabelas

Tabelas são criadas com a tag table e seguindo uma hierarquia parecida com a das listas.

Hierarquia das tabelas

Tabelas preciasm seguir essa ordem: Table > TR > TH > TD

  • Table = primeira tag pra criar a tabela
  • Table Row = linha da tabela
  • Table Header = cabeçalho
  • Table Data = conteudo

Depois tenho que colocar as linhas na tabela com a tag border, e para juntas as linhas uso a tag border-collapse com a função collapse.

Acrescentando classes no TR eu mudo a linha toda. Ja se acrescentada no TD mudo apenas uma parte especifica

Pra mudar o alinhamento vertical dos conteudos, uso a tag vertical-aling. A tag vertical-aling possui 3 funções, sendo elas:

  • Top = em cima
  • Middle = meio
  • Bottom = em baixo

Ex: pagina02

Tabelas grandes são simpes de criar, mas possuem uma semantica a ser seguida. Primeiro vem o Caption que é o titulo da legenda, depois vem o Head, o Body e terminam no Foot, suas respectivas tags são:

  • Caption = cria um titulo em cima da tabela
  • thead = nome dos conteudos da legenda
  • tbody = conteudos da tabela
  • tfoot = rodapé, conteudos finais ou conclusões da tabela

Mesmo que não acrescente nenhuma diferença visual na tabela, é excenssial colocar as tags scope:col pra avisar ao navegador que o conteuso são as linhas abaixo e scope:row para avisar que o conteudo são as linhas ao lado.

É possivel criar o efeito zebrado na tabela, basta usar a tag tbody > tr:nth-child(), e colocar entre os parentezes odd para as linhas impar e even para as linhas pares.

Ex: pagina03

Para colapsar células e preencher buracos na tabela. É possivel usar a tag rowspan e a quantidade que deseja preencher, e com cowspan é a mesma coisa, o que muda é que Rowspan mescla as células para baixo, e o Cowspan mescla para os lados.
