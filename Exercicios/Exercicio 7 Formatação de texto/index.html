Principais formatações de texto

Negrito / Destaque

Negrito é gerado usando a tag B

Também é possivel destacarem negrito usando a tag STRONG. Usar essa tag quando for dar sentido de importancia ou destaque em alguma palavra.

Itálico / Ênfase

Também da pra deixar em Itálico usando a tag I.

Em caso de destaque, é só usar a tag EM.

Texto marcado

Da pra cirar um texto marcado usando a tag MARK.

Caso queira trocar a cor da marcação sem usar as css, é só acrescentar a tag STYLE na tag MARK.

Texto grande e pequeno

Para deixar um texto grande, usa-se a tag BIG (pode ser que não funcione futuramente).

E para deixar o texto pequeno é com a tag SMAL

Texto exluido e texto inserido / sublinhado

Os textos inseridos / sublinhados são com a tag INS. Em caso de sentido é com a tag U.

Ja os textos excluidos são com a tag DEL. Usado geralmente pra destacar uma parte que deve ser lida e não considerada

Texto subescrito

Para coisas tipo x20+10, usa-se a tag SUP para subir o texto.

Texto sobrescrito

E para coisas tipo H2O usa-se a tag SUB para descer o texto.
