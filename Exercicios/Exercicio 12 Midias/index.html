Àudio no site

Para colocar áudio no site, usa-se a tag AUDIO, assim acrescentando CONTROL e AUTOPLAY

EX: Abertura de One Piece

Navegadores diferentes podem usar tipos diferentes de arquivo, tipo:

  1. MP3
  2. OGG
  3. WAV

Enserindo videos hospedados localmente

Para enserir videos, usa-se a tag VIDEO. O problema é que ele vem com o tamanho do video.

EX:

Semple acrescentar a tag CONTROLS para fazer aparecer o botão de play

Nos videos os formatos são:

  1. MP4
  2. MKV
  3. OGV

Para abrir diferentes tipos de video em diferentes navegadores, usa-se a tag SORCE dentro da tag VIDEO

também da pra colocar uma capa no video utilizando a tag POSTER

Videos hospedados no site com a tag VIDEO gastam muita internet e ficam caro pros clientes.

Comandos que podem ser adicionados a tag VIDEO são AUTOPLAY e LOOP
