Outras formatações de texto

Espaçamento nas letras

Quando for escrever alguma coisa e precisar de espaçamento nas letras, usa a tag CODE. Perfeito para escrever sites, porque deixa mais fácil de enxergar os caracteres e acentuações do link.

EX: Com a tag CODE fica assim Youtube.com, sem a tag CODE Youtube.com

Texto com quebra de linha

Para fazer aparecer no site exatamente como esta escrito no codigo usa-se a tag PRE 


    EX:
    DA
     PRA
      ESCREVER
       ASSIM

Citação

Quando for fazer uma citação entre aspas e com sentido, usa a tat Q

EX: Como diria o Zed A lamina que não se vê é a mais mortifera.

Citações completas

Obeserve sempre a atitude da cabeça, se quiser perceber a intensidade do dominio intelectual e espiritual daquela mente, naquele instante, aprovando ou rejeitando as emoções e instintos do seu corpo.

Abreviações

Agora da pra usar a tag ABBR para mostrar o que significa as palavras quando parar o cursor em cima do que deseja saber o significado. A partir de hoje, irei utilizar para fazer explicações das tags.

Texto invertido

Usando a tag BDO, da pra deixar o texto invertido.
