Formulário Básico

Sempre que criar um formulário, ele deve estar entre as tags FORM e /FORM. E para começar a criar os formulários, primeiro escolhe o nome da caixa que será preenchida, e para criar a caixa, uso a tag INPUT do tipo TEXT.

Para criar o botão de enviar, também é com a tag INPUT, só que com a configuração SUBMIT

Pra desligar a caixa de auto-completar, é só colocar na tag FORM a configuração AUTOCOMPLEAT:OFF

Configurações do formulário

O formulário presica ter algum lugar para ser enviado, então na tag FORM, coloco a configuração ACTION e escolher pra qual lugar vai (ex: cadastro.php)

Pra criar conecção entre o nome com a caixa dele, é só envelopar com a tag LABEL e informar o nome colocado no ID da caixa.
